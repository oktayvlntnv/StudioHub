# StudioHub Private

StudioHub Private is an owner-only personal streaming portal. Keep every feature legal, private, and explicit about source status.

## Safety Rules

- No piracy features.
- No torrent features.
- No DRM, login, token, or geo-block bypassing.
- No scraping protected streaming sources.
- No stream ripping, downloading, proxying, or unauthorized restreaming.
- No public registration, public sharing, or public source submission in the MVP.
- Never expose Xtream credentials to the frontend.
- Never log passwords or sensitive stream URLs.
- Imported M3U and Xtream items start as pending review.
- If legal playback is unclear, use `external_link` or keep the item hidden.

## Allowed MVP Work

- Mock frontend routes and UI.
- Mock metadata, posters, descriptions, genres, source labels, and legal statuses.
- YouTube official embeds when embedding is allowed.
- HTML5 playback only for public-domain, open-license, owned, or legally authorized streams.
- Source management UI that requires owner legal confirmation before import or approval.
